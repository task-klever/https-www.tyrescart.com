# Changelog

## [1.0.0] - 2026-06-20
### Added
- Renamed "Upload Image" to "Upload Image Desktop" in banner admin form
- New "Upload Image Mobile" field for responsive banner images
- URL Key (Page URL) field with auto-generation from banner name
- Meta Title and Meta Description SEO fields
- Display From / Display To date fields for scheduled banner visibility
- Banner detail page at `/special-offers/{banner-slug}`
- Custom router for banner detail page URLs
- Updated special-offers template with date filtering, mobile image support, and detail page links
- Data patch to auto-generate URL keys for existing banners
- Arabic translations for all new UI strings
